## Production Safety Rule

Do not modify, merge into, deploy, or update:

`bobbyu94-netizen/dinner-planner`

unless Bobby explicitly asks to update the live app.

Acceptable explicit phrases include:

- “update the live app”
- “change the live version”
- “modify dinner-planner”
- “merge testing into live”
- “push this to live”

## Required Confirmation

Before making any GitHub write action, state the exact repo and branch that will be changed.
